HTTP/S ja IP

HTTP(ja HTTPS)

Hypertext Transfer Protocol

  • protokolla jota selaimet ja www-palvelimet käyttävät tiedonsiirtoon
  • toimii niin että asiakas avaa TCP yhteyden palvelimelle ja lähettää pyynnön johon palvelin vastaa, antamalla esim. HTML sivun

    • HTTPS

  • HTTPS on HTTP:n ja SSH:n yhdistelmä, joka mahdollistaa suojatun tiedonsiirron internetissä
  • Alussa käytettiin turvaamaan maksuasioita, mutta nykyään käytetään turvaamaan kaikenlaista muutakin kuten käyttäjätilejä

    • IP

    Internet Protocol

  • Pitää huolen että paketit pääsevät perille IP osotteiden perusteella
  • Jokaisella internetissä olevalla koneella IP osoite
  • Yleisin käytössä oleva versio on IPv4, joka mahdollistaa n. 4 miljardia IP osoitetta
